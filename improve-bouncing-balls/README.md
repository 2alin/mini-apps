# Bouncing Balls (Improved)

This is my improvement to Mozilla's [Bouncing Balls Challenge](https://developer.mozilla.org/en-US/docs/Learn_web_development/Extensions/Advanced_JavaScript_objects/Adding_bouncing_balls_features).

## Try Live

Try the app: https://adilson.mx/mini-apps/improve-bouncing-balls/

Instructions: Use the arrow keys and/or the AWSD keys to move around and catch the balls.
